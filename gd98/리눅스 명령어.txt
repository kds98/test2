linux 명령어

man ls - 명령어 보기 q누르면 나가기 가능, 명령어 설명에 [] 없이 밑 줄만 있으면 생략 불가 무조건 적어야 함
ls -a - 파일 보기, . 현재 디렉토리의 이름 .. 상위 디렉토리 이름
ls -l - 파일 속성 보기, 세부적인 정보가 나온다
ls -F - 디렉토리 폴더 보기
ls -alf - 위 3개 합치기
ls -R - 서브 폴더까지 보기
mkdir - make directory, 폴더 만들기

cd /tmp/linux - 예제 파일들 존재

cp 파일명 . - 파일 카피
cat 파일명 - 파일 내용 보기
rm *.c - c로 끝나는 모든 파일 삭제
* - 모든

file 파일명 - 속성 보기

#include <fcntl.h>

vi 명령어
x - 하나 지우기
5x - 앞에 숫자만큼 지우기
. - 직전 명령 반복
